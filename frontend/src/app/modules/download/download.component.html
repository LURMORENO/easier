Extensiones navegador

Utilizar la extensión de EASIER es muy sencillo:

  1. Primero, descarga la extensión para tu navegador Google Chrome o Mozilla Firefox.
  2. Una vez instalado, visita una página web y selecciona el texto que desees.
  3. Luego, pulsa del botón derecho del ratón y selecciona la opción easier dentro del menú.
  4. Las palabras complejas aparecerán en azul.
