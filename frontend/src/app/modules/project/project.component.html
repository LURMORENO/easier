Proyecto

El proyecto EASIER (Más fácil) lo desarrolla el grupo de investigación Human Language and Accessibility Technologies (HULAT) del departamento de Informática de la Universidad Carlos III de Madrid (UC3M). El proyecto cuenta con la financiación de la empresa Indra y la Fundación Universia. El proyecto fue seleccionado como uno de los premiados en la “Convocatoria 2019 de ayuda a proyectos de investigación en Tecnologías accesibles Indra-Fundación Universia”.

El objetivo del proyecto es desarrollar soluciones tecnológicas que favorezcan el acceso a los contenidos web a las personas con discapacidad intelectual de manera específica, y a todas las personas en general.

Para ello, se ofrece un sistema web que favorece la accesibilidad cognitiva y proporciona diversas herramientas con recursos del lenguaje. La web está diseñada optimizando características de accesibilidad cognitiva siguiendo pautas de lectura fácil, lenguaje llano así como con las WCAG 2.1 (Nivel AA) para que sea accesible a todas las personas, incluyendo aquellas con discapacidad.

A este sistema web se puede acceder desde el ordenador o móvil. A partir de un texto en español introducido por el usuario, se detectan palabras complejas y se ofrecen sinónimos, definición y pictograma por cada palabra compleja detectada. Además, se ofrece una extensión para el navegador (Chrome y Firefox) que detecta las palabras complejas ofreciendo un sinónimo.

La detección de palabras complejas, el proporcionar sinónimos y definiciones se realiza utilizando métodos de Inteligencia Artificial como los del Procesamiento de Lenguaje Natural siguiendo un proceso sistemático.

Para las definiciones, se proporciona definición en lectura fácil o del diccionario de la Real Academia de la Lengua (RAE). Las definiciones de lectura fácil se obtienen del “Diccionario fácil” gracias a la colaboración con Plena Inclusión Madrid. Para proporcionar pictogramas, se utiliza los recursos ofrecidos por ARASAAC.
